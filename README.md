# Customer Segmentation Project 🚀

## 👨‍💻 Author
Aakash Jaiswal
  
## 📌 Overview
This project performs customer segmentation using multiple machine learning algorithms like:
- K-Means Clustering
- DBSCAN
- Agglomerative Clustering

## 🔧 Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📊 Features
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Multiple clustering techniques
- Visualization of clusters

## 🔥 Improvements Done
- Improved visualizations
- Added business insights
- Cleaned and structured dataset

## 💡 Business Insights
- High income + high spending → Premium customers
- Low income + low spending → Low value customers
- High income + low spending → Target customers
